Pengalaman Terhadap Python

Saya telah mengembangkan pemahaman dasar Python melalui berbagai latihan dan proyek kecil. Dari file-file yang saya kerjakan, saya memahami konsep dasar pemrograman Python seperti variabel, tipe data, operator, dan struktur kontrol (percabangan dengan if-else dan perulangan). Saya telah mempraktikkan pembuatan fungsi baik yang menggunakan parameter maupun yang tidak, termasuk fungsi dengan nilai kembalian (return). Contoh konkret termasuk latihan-latihan dasar seperti operasi matematika sederhana, manipulasi string, dan pengolahan input/output dasar.

Pengalaman saya mencakup penggunaan struktur data Python seperti list, tuple, dan dictionary. Saya memahami cara mengakses, menambah, mengubah, dan menghapus elemen dalam struktur data tersebut. Saya juga telah mempelajari konsep dasar pemrograman berorientasi objek (OOP) di Python termasuk pembuatan class, object, constructor, inheritance, dan method overriding. Dari file yang saya kerjakan, terlihat pemahaman saya dalam membuat modul sederhana dan menggunakannya melalui import statement baik secara langsung maupun dengan from-import.

Saya telah mempraktikkan operasi file handling dasar seperti membaca dan menulis file teks (file.txt). Untuk pengembangan lebih lanjut, saya terus mempelajari konsep-konsep menengah seperti exception handling, decorator, dan generator. Keterampilan Python saya saat ini mencakup pembuatan program kecil yang dapat menyelesaikan masalah-masalah komputasi dasar dan manipulasi data sederhana. Saya berencana untuk memperdalam pemahaman tentang struktur data yang lebih kompleks dan pengembangan aplikasi menggunakan framework Python.
