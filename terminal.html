Pengalaman Terhadap Terminal

Saya telah mengembangkan keterampilan dasar dalam menggunakan terminal Linux melalui berbagai praktikum dan eksperimen langsung. Dari materi yang saya pelajari, saya memahami struktur dasar perintah Linux yang terdiri dari command, option (dimulai dengan tanda -), dan argument. Saya terbiasa dengan perintah-perintah dasar seperti ls (dengan berbagai opsi seperti -l, -a, -F), cd, mkdir, cp, mv, dan rm untuk navigasi dan manajemen file. Saya juga telah mempraktikkan perintah sistem seperti date, cal, hostname, uname, who, dan whoami untuk memeriksa informasi sistem dan pengguna.

Pengalaman saya mencakup penggunaan manual (man) untuk mempelajari perintah baru, termasuk navigasi dalam tampilan manual menggunakan spasi, tombol panah, dan pencarian teks (/teks). Saya memahami pembagian bab manual dan bisa mencari perintah tertentu menggunakan apropos dan whatis. Selain itu, saya telah mempraktikkan manipulasi file teks dasar seperti melihat isi file (cat, more), mencari teks dalam file (grep), serta mengubah informasi pengguna dengan chfn. Saya juga familiar dengan konsep file hidden (diawali dengan .) dan bisa mengatur prompt shell menggunakan variabel PS1.

Untuk manajemen pengguna, saya memahami cara melihat identitas pengguna (id) dan mengkonfigurasi informasi finger. Saya terus mengembangkan keterampilan terminal dengan mempelajari perintah-perintah yang lebih advanced seperti pipeline (|), redirect (>, >>), dan manajemen proses. Pengalaman praktikum ini memberikan saya fondasi yang kuat untuk bekerja secara efisien di lingkungan command line Linux.
