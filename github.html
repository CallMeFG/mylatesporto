Pengalaman Terhadap Github

Saya memiliki pengalaman dasar dalam menggunakan GitHub untuk manajemen kode sumber pribadi. Saya mampu membuat repository baru, mengupload file melalui antarmuka web GitHub, dan melakukan editing sederhana pada file-file proyek. Saya memahami cara membuat repository menjadi publik atau privat, serta mengatur izin akses dasar untuk berbagi repository dengan orang lain secara kolaboratif. Selain itu, saya telah berhasil mendeploy website statis sederhana menggunakan GitHub Pages dengan mengaktifkan fitur tersebut melalui pengaturan repository.

Keterampilan saya mencakup penggunaan perintah Git paling dasar seperti commit dan push melalui GitHub Desktop, meskipun saya belum mahir menggunakan command line. Saya bisa melihat riwayat perubahan file dan kembali ke versi sebelumnya jika diperlukan. Untuk kolaborasi, saya baru sebatas bisa memberikan akses repository ke orang lain. Saya terus belajar untuk meningkatkan pemahaman tentang workflow Git yang lebih terstruktur.
