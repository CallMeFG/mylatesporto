Pengalaman Terhadap HTML

Saya telah mempelajari dan menguasai berbagai konsep dasar HTML melalui platform W3Schools, dimulai dari pengenalan struktur dokumen HTML, penggunaan editor, hingga penerapan elemen-elemen fundamental seperti headings, paragraphs, dan lists. Pemahaman saya mencakup cara menyusun kerangka website dengan tag dasar seperti <html>, <head>, dan <body>, serta memanfaatkan atribut untuk menyesuaikan perilaku elemen. Selain itu, saya juga terbiasa dengan teknik pemformatan teks (formatting) seperti tebal, miring, atau garis bawah menggunakan tag <b>, <i>, dan <u>, serta penggunaan quotations dan comments untuk meningkatkan keterbacaan kode.

Selain elemen dasar, saya telah menguasai fitur lanjutan seperti pembuatan links dengan tag <a>, penyisipan gambar melalui <img>, dan pengaturan favicon untuk identitas visual website. Saya juga memahami pentingnya pengelolaan tata letak menggunakan block (seperti <div>) dan inline elements, serta penerapan classes dan id untuk styling dengan CSS. Keterampilan saya mencakup pembuatan tabel (<table>) untuk menampilkan data terstruktur dan daftar (<ul>, <ol>) untuk konten berpoin. Tidak ketinggalan, saya mempelajari integrasi JavaScript dengan HTML dan penggunaan iframes (<iframe>) untuk menyematkan konten eksternal.

Pengalaman saya juga meliputi optimasi tampilan dengan CSS dasar, seperti mengatur warna (colors), font, dan layout melalui atribut style atau file terpisah. Saya paham cara mengelola file paths untuk mengorganisir sumber daya (gambar, CSS, JS) dan teknik head untuk metadata seperti judul halaman (<title>) atau karakter encoding. Dengan pemahaman menyeluruh ini, saya siap mengembangkan website portofolio yang interaktif, terstruktur, dan responsif, sekaligus terus memperdalam topik lanjutan seperti HTML5 semantik atau framework pendukung.
