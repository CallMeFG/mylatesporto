Pengalaman Terhadap Figma

Saya telah mengikuti bootcamp Figma intensif selama 2 minggu yang fokus pada penguasaan tools dasar desain antarmuka. Dari pelatihan tersebut, saya memahami cara membuat layout sederhana menggunakan frame, shape tools (ellipse, rectangle), dan mengatur hierarki visual melalui layers panel. Saya mampu bekerja dengan berbagai assets seperti import gambar (contoh: handphone, globe) dan mengorganisir komponen dalam group untuk efisiensi workflow. Contoh konkret termasuk proyek desain aplikasi mobile sederhana dengan elemen-elemen dasar seperti tombol, search bar, dan kategori menu.

Pengalaman praktik saya mencakup pembuatan ilustrasi dasar dengan shape tools dan penanganan teks (typography) untuk konten desain. Saya telah mempraktikkan teknik grading warna dan export desain ke berbagai format. Dalam proyek tim, saya berkontribusi pada desain halaman aplikasi dengan komponen seperti 'Top up & Tagihan', 'Kategori Pilihan', dan elemen navigasi sederhana. Saya juga memahami prinsip penamaan layer yang terstruktur (contoh: Group 3, Group 5) untuk memudahkan kolaborasi.

Melalui modul khusus, saya belajar membuat desain feed Instagram dengan memanfaatkan fitur auto layout dasar dan style consistency. Saya terbiasa dengan workflow Figma mulai dari pembuatan frame artboard, penempatan assets, hingga export final design. Untuk proyek kolaboratif sederhana, saya dapat bekerja dalam file bersama dengan tim menggunakan fitur komentar dasar, meskipun belum menguasai fitur prototyping atau design system yang lebih kompleks.

Saat ini, keterampilan Figma saya mencakup pembuatan wireframe statis dan high-fidelity design untuk aplikasi mobile sederhana dengan komponen dasar. Saya terus berlatih untuk meningkatkan pemahaman tentang auto layout yang lebih advance, komponen variants, dan basic prototyping untuk interaksi sederhana. Pengalaman bootcamp telah memberi saya fondasi kuat untuk mengembangkan skill UI/design system yang lebih terstruktur di masa depan.
