Pengalaman Terhadap MySql

Saya telah mengembangkan keterampilan yang kuat dalam MySQL, khususnya dalam menulis dan mengoptimalkan query SQL untuk berbagai operasi database. Dari contoh-contoh yang saya kerjakan, saya memahami dengan baik operasi dasar seperti SELECT, INSERT, UPDATE, dan DELETE, serta fungsi agregasi termasuk COUNT, SUM, AVG, MIN, dan MAX. Saya telah mempraktikkan penggunaan klausa WHERE untuk filtering data, GROUP BY untuk pengelompokan, dan HAVING untuk memfilter hasil agregasi. Selain itu, saya terbiasa dengan pengurutan data menggunakan ORDER BY (baik ascending maupun descending) dan pembatasan hasil query dengan LIMIT. Contoh konkret termasuk query untuk menemukan karyawan dengan gaji tertinggi/terendah, menghitung rata-rata gaji per departemen, dan menampilkan data dengan kriteria spesifik.

Pengalaman saya juga mencakup penggabungan tabel (JOIN) yang kompleks, termasuk INNER JOIN, LEFT JOIN, dan subquery untuk mengakses data dari beberapa tabel yang berelasi. Saya telah membuat query yang menghubungkan tabel employees, departments, dan job_history untuk mengekstrak informasi seperti riwayat pekerjaan karyawan atau daftar departemen dengan gaji tertinggi. Pemahaman saya tentang alias tabel (table aliasing) membantu dalam menyederhanakan query yang melibatkan banyak tabel. Saya juga telah mempraktikkan penggunaan subquery baik di klausa WHERE maupun FROM, seperti mencari karyawan dengan gaji di atas rata-rata departemennya atau menampilkan data departemen yang memiliki karyawan dengan kriteria tertentu.

Pada level desain database, saya memiliki pengalaman dalam membuat diagram relasi entitas (ERD) yang mencakup entitas seperti semester, matkul, professor, dan relasi antar entitas tersebut. Saya memahami konsep normalisasi database hingga bentuk normal ketiga (3NF) dan dapat menerapkannya dalam desain skema database. Dari diagram yang saya buat, terlihat pemahaman tentang kardinalitas relasi (one-to-one, one-to-many, many-to-many) dan penggunaan primary key/foreign key untuk menjaga integritas referensial. Saya juga familiar dengan pembuatan view untuk menyederhanakan akses data yang kompleks dan penggunaan index untuk meningkatkan performa query.

Untuk operasi database yang lebih maju, saya telah mempelajari teknik transaksi untuk menjaga konsistensi data, serta dasar-dasar stored procedure dan trigger. Saya juga memahami pentingnya keamanan database dengan menerapkan hak akses yang tepat dan sanitasi input untuk mencegah SQL injection. Pengalaman saya mencakup optimasi query dengan EXPLAIN untuk menganalisis execution plan, serta teknik indexing strategis untuk mempercepat query pada tabel besar. Saya terus mengembangkan keterampilan ini dengan mempelajari fitur-fitur MySQL yang lebih canggih seperti window functions, CTE (Common Table Expressions), dan partisi tabel.

Secara keseluruhan, saya memiliki fondasi yang kuat dalam SQL dan administrasi database MySQL. Kemampuan saya mencakup dari desain database, penulisan query kompleks, hingga optimasi performa. Saya siap menerapkan keterampilan ini dalam pengembangan aplikasi berbasis database yang membutuhkan manipulasi data yang efisien dan andal. Kedepannya, saya berencana untuk memperdalam pengetahuan tentang database clustering, replication, dan teknik backup/recovery untuk skala enterprise.
